# 🚀 AI ATS Resume Score Checker

An AI-powered resume analysis platform that helps job seekers optimize their resumes and improve their chances of getting shortlisted by Applicant Tracking Systems (ATS).

## 📌 Overview

AI ATS Resume Score Checker analyzes resumes against job descriptions, calculates ATS compatibility scores, identifies missing keywords, and provides personalized recommendations to improve resume quality and job-fit.

The platform is designed for both **job seekers** and **recruiters**, making the hiring process smarter, faster, and more efficient.

---

## ✨ Features

- 📄 Resume Upload (PDF)
- 🎯 ATS Compatibility Score
- 🔍 Job Description Matching
- 🧠 AI-Powered Keyword Analysis
- 📊 Resume Quality Evaluation
- 💡 Resume Improvement Suggestions
- 📈 Skill Gap Analysis
- 👨‍💼 HR Candidate Screening Support
- ⚡ Fast and User-Friendly Interface

---

## 🎯 Problem Statement

Many qualified candidates are rejected before reaching recruiters because their resumes fail ATS screening due to:

- Missing keywords
- Poor formatting
- Weak project descriptions
- Low relevance to job requirements

This platform helps candidates identify and fix these issues before applying.

---

## 💡 Solution

The AI ATS Resume Score Checker:

1. Extracts content from uploaded resumes.
2. Compares resumes with job descriptions.
3. Calculates ATS and job-fit scores.
4. Identifies missing keywords and skills.
5. Generates actionable improvement suggestions.
